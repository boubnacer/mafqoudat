Instructions for Live App Testing

  1. Open your React app in the browser
  2. Open DevTools Console (F12)
  3. Check localStorage: localStorage
  4. Verify NO theme key exists
  5. Verify globalState exists with mode property
  6. Click the theme toggle button in the app
  7. Verify the theme changes visually
  8. Check localStorage.globalState and see mode changed
  9. Refresh the page
  10. Verify theme persists
